The Basic Language of the Web: HTML

Posted by Laura Jones on Mondya, June 21 2027 lorem60

All modern websites and web applications are built use three fundamental technologies: HTML, CSS and Javascript. These are the languages of the web

In this post, lets focus on HTML. WE will learn what HTML is all about, and why you shoudl learn it.

What is HTML?

HTML stands for Hyper Text Markup Language. It's the markup language that web developers use to structure and describe the content of a webpage(not a programmign language).

HTML consists of elemnets tahtd escribe different types of content: paragraphs, links, headings, images, video, etc. Web browsers understand HTML and render HTML code as websites

In HTML, each element is made up of 3 parts:

  1. The opening tag
  2. The closing tag
  3. The actual element

You can learn more on MDN Web Docs

Why should you lern HTML?

There are countless reasons for learning the fundamental language of the web. Here are 5 of them:

  • To be able to use fundamental web dev language
  • To handcraft beautiful websites instead of relying on tools liek Wordpress or Wix
  • To build web application
  • To impress friends
  • To have fun
